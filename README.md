# vue2-vuecli-demo

## Project setup

```
yarn install
```

### Compiles and hot-reloads for development

Windows:

```
$Env:NODE_OPTIONS='--openssl-legacy-provider'
yarn serve
```

Unix:

```
yarn serve
```

### Compiles and minifies for production

```
yarn build
```

### Lints and fixes files

```
yarn lint
```
